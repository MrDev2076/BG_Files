String Manipulation Challenge

Problem Statement

You are given a sentence containing both uppercase and lowercase letters.

Your task is to perform the following operations:

  1. Convert the entire sentence to lowercase.
  2. Check if the sentence starts with the word "java" (case-insensitive).
  3. Replace all occurrences of the word "world" with "universe."
  4. Extract the substring from the 5th to the 10th character (inclusive).
  5. Count the number of occurrences of the letter 'a' in the modified sentence.
  6. Print the final modified sentence.

Sample Input

inputSentence = "Hello Java World! Java is amazing."

Sample Output

Modified Sentence: "java is amazing. java is amazing."

Substring (5th to 10th): "is ama"

Occurrences of 'a': 8

Explanation

Convert to lowercase: "hello java world! java is amazing."

Starts with "java": Yes

Replace "world" with "universe": "hello java universe! java is amazing."

Extract substring (5th to 10th): "is ama"

Count occurrences of 'a': 8

Print the modified sentence: "java is amazing. java is amazing."
